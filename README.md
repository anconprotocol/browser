### Codename El Segundo


## A browser / aggregator aka universal wallet but for metaverse. WalletConnect only

### Tech stack for the rookies

- ParkyDB: Dexie, IPLD DAG, Waku Connect (websockets)
- Waku v2 RPC
- RxJS

### Required nodes

- Ancon Node
- Waku Node

### Works with

- WalletConnect
- EVM chains
